# Gerador de Backgrounds para Cartazes de Oferta

Aplicação em React + Vite que cria automaticamente fundos promocionais responsivos. As variações são pensadas para uso em peças comerciais (cartazes de oferta, posts, stories) e funcionam em qualquer proporção.

## Recursos
- Combina gradientes, padrões e elementos decorativos para gerar centenas de composições exclusivas.
- Ajuste de proporção (1:1, 4:5, 9:16 e 16:9) com adaptação automática de elementos visuais.
- Controle do número de variações exibidas simultaneamente.
- Interface em português pensada para equipes de marketing e varejo.

## Executando localmente

```bash
cd offer-background-generator
npm install
npm run dev
```

O servidor de desenvolvimento será iniciado em `http://localhost:5173` (porta padrão do Vite).

Para gerar uma versão otimizada, execute:

```bash
npm run build
```
